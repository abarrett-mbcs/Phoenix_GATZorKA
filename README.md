# Phoenix_GATZorKA

"You shouldn't need a $3,000 M3 Max to learn Kubernetes. You just need a strategy."

Project Phoenix_GATZorKA is a declarative blueprint for the modern SRE. This isn't just a collection of scripts; it's a Phoenix-style resurrection of a 15-year-old laptop.

The Stack:
GitHub (The Source of Truth) – Version control and Agile project management via GitHub Projects.

Ansible (The Architect) - Orchestrating the OS state.

Terraform (The Foundation) – Declarative provisioning of the cloud-side resources

ZorinOS (Hybrid Cloud Control Plane) - This will be the core, the Operating System of the repurposed 2011 Macbook Pro.

K3d (The Cluster) - Lightweight Multi-node K8s.

ArgoCD (The Heart) - True GitOps automation.

- - - 

Docker (The Engine) - Powering the container runtime.

K9s (The Cockpit) - Terminal UI for cluster management.

Lens (The Radar) - Visualizing the workload.
